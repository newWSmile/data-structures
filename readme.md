5 时间频度

算法中的执行次数称之为时间频度 记做T(n)，特点：

1）忽略常数项

2）忽略系数

3）忽略低次项

6 时间复杂度

1)用1 代替运行时间中的所有加法常量

2)修改后的运行次数函数中，只保留最高阶

3)取出最高阶的系数

example:

常数阶：O(1)        

对数阶: O(log₂N)

线性阶: O(n)

线性对数阶: O(nlogN)

平方阶: O(n²)
